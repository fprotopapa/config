|Shortcut | Description |
|-----------|---------|
|CTRL + z  |
| \| | Horizontal split |
| - | Vertical split |
| Arrows | Switch Pane |
| c | Create new window |
| n | Switch window |
| # | Switch to window # |
| d | Detach from session |
| s | Switch btwn. sessions |
| , | Rename window |
| :kill-session | Close session |
| tmux |
| ls | List sessions |
| a | Attach to session |
| a -t # | Attach to session # |
| new -s \<n> | Create session \<n> |  
| source ~/.tmux.conf | Load new config |
