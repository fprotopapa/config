# Picocom
```
sudo apt-get install picocom

picocom -b 115200 /dev/tty<con>
```
- https://linux.die.net/man/8/picocom

| Shortcut | Description |
| -------- | ----------- |
| CTRL + a  | 
| CTRL + x  | Exit |
| CTRL + s | Send file |
| CTRL + r | Receive file |
