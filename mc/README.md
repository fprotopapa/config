# Midnight Commander
|Shortcut | Description |
|-----------|---------|
|CTRL + s | Quick search |
|TAB | Switch Panel |
|+ | Select by pattern |
|Shift + arrow | Select |
|F6 | Move files |
|ALT + c | Quick cd (e.g. sftp://user@machine) |
|CTRL + \ | Directory Hotlist |
|On select: |
|CTRL + Space | Folder size |
